## 阴阳之道
```text
嗜欲深者天机浅，嗜欲浅者天机深
人要经常反省
太极生两仪，两仪生四象
乱中有序
自天佑之，吉无不利
孔子：一阴一阳之谓道（自然孕育），易有太极，是生两仪（阴阳）
一切都是有例外的，
一分为二（西方的观点）
一生二，二生三，三生万物（道家的观点）
```
